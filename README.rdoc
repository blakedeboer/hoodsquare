What is the Williamsburg of San Francisco? 

Rails app that finds similar neighborhoods in different cities based on Foursquare data and user input. 

Leveraged the K-Means algorithm to measure the similarity of a neighborhood in one city to all the neighborhoods in another city and allows users to vote on the results which allows the app to "learn"

Technologies used:
* Foursquare API
* Foursquare 2 gem
* K-means gem
* Flickr API 
* AJAX
* jQuery
* Bootstrap
